# Breast Cancer Prediction

## Project Overview
This project predicts breast cancer diagnosis using machine learning classification algorithms.

##  Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib

##  Project Steps
1. Data Loading
2. Data Preprocessing
3. Exploratory Data Analysis (EDA)
4. Model Training
5. Model Evaluation

##  Models Used
- Logistic Regression
- Decision Tree Classifier

## Results
The models were evaluated using accuracy score, confusion matrix, and classification report.
